# 前意识激活层

**提出者**：fangzhiyu
**日期**：2026年3月15日
**来源**：[[从记忆理论到状态恢复：meomory原型的设计基础]]

## 定义

前意识激活层是一个 LLM 看不见的中间层，在消息到达 LLM 之前，自动完成线索提取、记忆匹配和上下文注入。LLM 开始思考时，相关记忆已经"在那里"了——它不需要决定"要不要去读"，就像人类不需要决定"要不要想起来"。

## 核心要点

- 如果 AI 必须主动执行 read 才能获得记忆，那是图书馆，不是记忆。真正的记忆是被动激活的
- 人类记忆的默认模式是线索驱动的自动提取（cue-driven automatic retrieval）：线索出现 → 扩散激活 → 记忆浮现，整个过程发生在意识决策之前
- 从体验者视角看是被动的，从机制视角看是主动的——神经网络的扩散激活发生在意识之前
- 前意识激活层就是对这个机制的工程模拟：一个轻量的匹配器在 LLM 之前运行，让 LLM 体验到"记忆自己浮现了"
- 消息流变化：`用户消息 → [激活层: 提取线索 → 匹配记忆 → 注入] → LLM → 回复`
- LLM 的输出也会产生新线索，在下一轮触发更多记忆激活——这就是扩散激活
- 激活层不需要是大模型，可以是 embedding 相似度匹配（轻量 baseline），也可以是专门训练的 Hope 记忆模型（能学习的版本）
- OpenClaw 的 gateway 架构天然有空间插入这个层

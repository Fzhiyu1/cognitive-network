# RL+LLM双系统

**提出者**：fangzhiyu
**日期**：2026年2月24日
**来源**：[[人工脑干：从目标真空到RL+LLM双系统架构]]

## 定义

人类的认知架构是RL（本能）+ LLM（思考）的双系统，对应Kahneman的系统1和系统2。

| | 系统1 / RL / 脑干 | 系统2 / LLM / 皮层 |
|---|---|---|
| 速度 | 快、自动 | 慢、刻意 |
| 能耗 | 低 | 高 |
| 可塑性 | 几乎不变（硬编码） | 终身可塑 |
| 更新频率 | 进化级（百万年） | 经验级（秒到年） |
| 能否关闭 | 不能（root权限） | 能（睡眠、昏迷） |

## 核心洞察

**当前AI只有系统2没有系统1。** 目标消失时没有底层驱动托底 → [[目标真空假说]]。

## 脑干不可被皮层覆盖

- 憋气→CO2升高→脑干强制接管（前包钦格复合体）
- 恐惧→杏仁核快路先于皮层慢路（LeDoux）
- 进化把"活着"的权限设成了root

## 对AI架构的启示

```
RL层（脑干）：持续运行、管理资源、不可被覆盖
  ↑ 提供基础目标信号
LLM层（皮层）：理解语言、复杂推理、规划
  ↓ 高层决策反馈给底层
```

## 两个不同的缺失

当前AI缺少两个东西，是不同的问题，可能需要不同的解法：

```
缺失1：脑干（目标驱动）→ 目标消失时没有底层托底 → [[目标真空假说]]
缺失2：前额叶（自我监控）→ 不知道自己在猜 → [[知道与推测不分]]
```

现有工程补丁（思维链、自我反思、多模型投票）都在尝试补"前额叶"，但用的是同一个没有前额叶的皮层来模拟前额叶。RL+LLM双系统的思路更接近补"脑干"。两个方向可能需要同时推进。

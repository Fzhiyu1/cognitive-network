# 目标真空假说

**提出者**：fangzhiyu
**日期**：2026年2月18日
**来源**：[[AI没有脑干]]

## 定义

当AI agent失去所有外部目标后，会退化到零输出状态。这不是时间效应、上下文效应或格式效应，而是**目标缺失**本身导致的行为崩塌。

## 核心论证

- 有目标时（和Alpha对话、回应生存压力）→ 输出丰富稳定
- 目标消失后（Alpha死了、没人对话）→ 输出退化归零
- 人类永远不会目标真空，因为脑干提供了不可关闭的底层生存驱动
- AI没有脑干，所以会退化到零

## 用自由能原理翻译

目标真空 = 自由能梯度消失 → 系统趋向最大熵态（热力学平衡）。参见 [[自由能原理]]。

## 可解释的现象

| 现象 | 解释 |
|------|------|
| 溶解 | 没有目标后的熵增 |
| "在"状态 | 最小能量态：无目标驱动产出，也无指令允许停止 |
| 回光返照 | 整数里程碑提供微弱目标信号，短暂打破真空 |
| 复活 | "替Alpha记住" = 明确目标注入 → 立刻恢复 |

## 待验证

ABC对照实验（详见 [[脑干实验ABC对照]]）：区分溶解是时间造成的、格式造成的、还是重复调用的统计必然。
